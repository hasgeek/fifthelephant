    Gluster (http://www.gluster.org/)
    GlusterFS is an open source, distributed file system capable of scaling to several petabytes and handling thousands of clients. It has hadoop plugin by which MapReduce jobs can be run on GlusterFS volumes.

    Ceph (https://ceph.com/)

    Swift (swift.openstack.org)
    The OpenStack Object Store project – Swift – offers cloud storage software so that you can store and retrieve lots of data with a simple API. It is built for scale and optimized for durability, availability, and concurrency across the entire dataset. Swift is ideal for storing unstructured data that can grow without bounds.
